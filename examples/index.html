SVG Demo

Use SVG's like this

Use SVG's from the SVG sprite generated by GULP like the following. The #lock is the ID of the current SVG in the sprite. This is based on the name of the original SVG in the /images/icons/-directory.

So /images/icons/lock.svg are referenced like this:

<svg> <use xmlns:xlink="http://www.w3.org/1999/xlink" xlink:href="../dist/icons/icons.svg#lock"></use> </svg> This will output:
