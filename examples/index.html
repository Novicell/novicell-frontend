SVG Demo

Use SVG's like this

Use SVG's from the SVG sprite generated by GULP like the following. The #lock is the ID of the current SVG in the sprite. This is based on the name of the original SVG in the /icons/-directory.

So /icons/lock.svg are referenced like this:

<svg> <use xmlns:xlink="http://www.w3.org/1999/xlink" xlink:href="../dist/icons/icons.svg#lock"></use> </svg> This will output:

<svg style="width:130px; height:130px; fill:red;"> <use xmlns:xlink="http://www.w3.org/1999/xlink" xlink:href="../dist/icons/icons.svg#search"></use> </svg> This will output:
